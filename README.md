# 使用说明
--app 代码运行目录    
--movie_env 代码运行环境   
1、source movie_env/bin/active         
2、运行mysql数据库，在app/__init__.py里面配置数据库账号和密码     
3、python manage     #运行程序    
4、打开浏览器输入127.0.0.1
